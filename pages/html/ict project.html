HNDIT1052 ICT Project (Individual)


General Course Information

Course Details

Course Code : HNDIT1052
Course Title : ICT Project (Individual)
Diploma Program : HNDIT
Semester : 1
Course Status : Compulsory, GPA
Number of Credits : 3
Mode of Delivery :Lectures, Discussion, Presentations, Demonstrations, LMS, e learning, Assignment, Practical, blended learning, independent learning, Guided learning, Portfolio, Logbook

Assumed Background : This is the 1st year 1st semester Practical base course module. The required background is their A/L theory and practical knowledge and self-learning capacity to apply their basic ICT knowledge to a simple complete project.

Course Introduction: This course module covers how to apply Basic office package knowledge to proposal writing, proposal presentation, project report writing, project implementation etc. They can select a small application project in Excel, Access, Arduino, Micro BIT, Graphics & Animation.

Assessment Plan

Assessment Summary
Supervision of Project (Individual): Lecturer who supervises 5 projects may allocate 1 hour (Max: 0 3 Hours)

Type Activity Weighting
Continuous Assessment Project Proposal 10%
Proposal presentation 10%
Project Budget 10%
Interim presentation/ Poster presentation/ Evaluating Blog 20%
Final Presentation 30%
Final Presentation 20%
Total 100%


Teaching and learning Strategies

One hours of lectures held every week will be the main method used to explain the theoretical component whereas another four hours per week will be utilized for the project works within laboratories.
