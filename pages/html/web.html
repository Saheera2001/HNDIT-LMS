HNDIT1022 Web Design


General Course Information

Course Details

Course Code : HNDIT1022
Course Title : Web Design
Diploma Program : HNDIT
Semester : 1
Course Status : Compulsory/GPA
Number of Credits : 4
Mode of Delivery : Lectures, Discussion, Presentations, Assignment, Practical, blended learning, Demonstrations, LMS, e learning, independent learning, Guided learning, Portfolio
Assumed Background :A/L theory knowledge

Course Introduction: This course introduces students to basic web design using HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets). The course does not require any prior knowledge of HTML or web design.
